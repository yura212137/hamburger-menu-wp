概要
- このサイトはオンラインスクールRaiseTechの課題である、架空のハンバーガーショップのサイトです。

使用プラグイン
- WP-PageNavi
    投稿一覧、検索ページのページングに使用
- Custom Field Suite
    フロントページのTakeOut、EatIn、AccessにHP管理者が表示内容を変更できるようにカスタム投稿タイプを追加。
    投稿内容をカスタムフィールドを使用することでデザインが崩れることを防止する。
    やりたかったができなかったこと
    AccessのカスタムフィールドにgooglemapのURLを入力できるようにしたかった。

使用例
- メニューの追加があれば投稿から入力
- サイドバーメニューは管理画面から追加、削除
- フロントページのTakeOut、EatInには管理画面のお知らせから表示内容を変更、追加、削除
- Accessも上記同様にお知らせから表示内容を変更
